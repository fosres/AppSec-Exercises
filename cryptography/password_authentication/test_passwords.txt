# Hash Format Details (PHC Standard)
# ===================================

All hashes use PHC (Password Hashing Competition) standard format,
similar to hashes in /etc/shadow on Unix systems.

Format: $algorithm$parameters$salt$hash

Examples:
- scrypt:   $scrypt$ln=17,r=8,p=1$...$...
- pbkdf2:   $pbkdf2-sha256$rounds=600000$...$...
- argon2:   $argon2id$v=19$m=19456,t=2,p=1$...$...
- bcrypt:   $2b$12$...[salt+hash]...

All parameters, salt, and hash are encoded in the string.
No need for custom parsing - passlib handles everything!
